# Quantumult X Rules

Personal Quantumult X rule resources.

## Raw URLs

After pushing this folder to GitHub, use a raw URL like:

```text
https://raw.githubusercontent.com/<username>/<repo>/main/quantumult/claude.list
```

Quantumult X example:

```conf
https://raw.githubusercontent.com/<username>/<repo>/main/quantumult/claude.list, tag=Claude, force-policy=美国节点, update-interval=172800, opt-parser=true, enabled=true
```

For the full AI bundle:

```conf
https://raw.githubusercontent.com/<username>/<repo>/main/quantumult/ai-us.list, tag=AI-US, force-policy=美国节点, update-interval=172800, opt-parser=true, enabled=true
```
